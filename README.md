## üìú How to Get Started:

1. **Clone the Repository:**
```
git clone https://github.com/devlarabar/whats-the-tea.git
```

2. **Install Dependencies:**
```
cd whats-the-tea
npm install
```

3. **Run the Application:**
```

```

5. **Open Your Browser:**
Visit `http://localhost:3000`.
